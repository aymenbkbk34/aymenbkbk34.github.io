MediaTek86 — Gestion des documents de la médiathèque

Contexte : Evolution d'une application de bureau C# exploitant une API REST PHP pour gérer les documents des vidéothèques de la chaîne MediaTek86.

Cadre : Atelier de professionnalisation (AP)

Technologies utilisées

  • C# / .NET Framework 4.7.2
  • PHP / API REST
  • MySQL
  • Visual Studio 2022
  • NetBeans
  • WampServer
  • GitHub

Fonctionnalités ajoutées

  • Gestion des commandes de livres et DVD
  • Gestion des commandes de revues (abonnements)
  • Système d'authentification avec gestion des droits
  • Sécurisation des credentials
  • Intégration des logs
  • Tests unitaires
  • Documentation technique
